emotion.valence: -1.0 (very negative) to 1.0 (very positive)
emotion.arousal: 0.0 (calm) to 1.0 (excited/agitated)
emotion.dominant: The strongest single emotion
emotion.trigger: What caused this emotion (brief description)
emotion.concern_level: 0=none, 1=monitor, 2=alert (crisis signals)

fragment.detected: true if this conversation turn contains story-worthy content
fragment.elements.when: Time period mentioned (null if not stated)
fragment.elements.where: Place mentioned (null if not stated)
fragment.elements.who: People mentioned (empty array if none)
fragment.elements.what: What happened (null if not a specific event)
fragment.elements.emotion: Emotion attached to the story (null if none)
fragment.elements.why: Why it matters to the user (null if not stated)
